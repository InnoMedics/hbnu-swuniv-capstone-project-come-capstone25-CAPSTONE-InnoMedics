## <u>Teamate</u> Project Background
- ### 필요성
  - 병리학 연구에서 췌장 베타세포 분석은 질병 진행도를 평가하기 위한 핵심 과정이지만, 현재는 대부분 수작업(ImageJ 등)에 의존하고 있음.
  - 연구자 개인의 주관적 판단에 따라 결과가 달라지고, 수백 장의 이미지를 분석하는 데 장시간이 소요되어 비효율적임.
  
- ### 기존 해결책의 문제점  
  - 수작업 기반의 라벨링 및 분석 과정은 개인별 데이터 일관성을 유지하기 힘들며,
  대규모 병리 영상 연구에 많은 시간이 소모된다.
  
## System Design
  - ### System Requirements
    - 독립 실행형 딥러닝 분석 도구: 인터넷 환경에 의존하지 않고, 로컬 환경에서 모델 실행 및 결과 시각화가 가능해야 함.
    - U-Net 기반 이미지 분석 모델: 베타세포 영역을 자동 분할하고, 면적 기반의 정량 분석 결과 제공.
    - MFC 기반 GUI: 의료진 및 비전공자도 손쉽게 사용할 수 있도록 직관적인 사용자 인터페이스 제공.
    
## Case Study
  - ### Description
    충남대학교 병원에서 제공한 쥐 췌장 병리 이미지(염색 이미지)를 데이터셋으로 사용하였으며,
  U-Net 모델을 학습시켜 베타세포 영역을 자동으로 검출하도록 설계했다.
  학습 과정에서는 Albumentations를 이용한 이미지 증강과 IoU/Dice Score 기반 평가 지표를 사용하여 모델의 일반화 성능을 개선했다.

   GUI(MFC) 상에서는 사용자가 이미지를 선택 후 “분석 실행” 버튼을 누르면,
  U-Net 모델로 분석을 수행하고 결과 마스크를 생성한다.
  분석된 결과는 시각적으로 확인할 수 있으며, 로컬상으로 자동 저장된다.
 
  
## Conclusion
  - ### 프로젝트 요약

    본 연구는 병리 이미지를 자동으로 분석하고, 연구자의 주관적 편차를 줄이는 딥러닝 기반 분석 자동화 도구를 구축하였다.
    특히 수작업 중심이던 기존 분석 과정을 자동화함으로써 분석 시간 단축과 객관성 확보를 실현하였다.

  - ### 기대효과

    - 의료 및 생명과학 연구자의 데이터 분석 효율 향상
    - 의료 영상 자동 분석 기술의 상용화 기반 마련
    - 다른 조직(간, 폐 등) 분석으로의 확장 가능성 확보
